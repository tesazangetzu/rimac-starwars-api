# rimac-starwars-api

## .env example

```
PORT=3000
ENVIROMENT=local
API_URL=https://swapi.py4e.com

DB_HOST=
DB_USER=
DB_PASSWORD=
DB_DATABASE=starwars_api
```

## Setup:

1. Install dependencies

> npm install

2. Create and configure .env

3. Run migrations

> npm run migration

4. Start server

> nodemon src/app.js

or

> npm run dev

### Note: Before starting project deploy a database for mysql at RDS or local enviroment
